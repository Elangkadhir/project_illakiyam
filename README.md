📜 Project Illakiyam

Project Illakiyam is a modern digital platform that brings the richness of ancient Tamil literature—especially Sangam-era poems—into a clean, user-friendly interface.

This project aims to preserve, explore, and experience timeless Tamil works by transforming classical texts into an accessible and engaging digital format.

✨ Features
📖 Collection of Sangam literature and classical Tamil poems
🌐 Tamil + English translations for better understanding
🧠 Multiple commentaries (urai) from different scholars
🔍 Easy navigation by Paal, Iyal, and Athigaram
📱 Responsive and modern UI design
⚡ Fast and optimized static data (JSON-based)
🎯 Goal

To make Tamil literature more accessible to the modern generation by combining traditional content with modern UI/UX, ensuring that the beauty and wisdom of Sangam literature are preserved and appreciated worldwide.

🛠️ Tech Stack
React.js / Vite
Tailwind CSS
JSON-based content structure
❤️ Inspiration

Inspired by the timeless works of Tamil scholars and poets, this project is a step towards digitizing and celebrating one of the world’s oldest living languages.
